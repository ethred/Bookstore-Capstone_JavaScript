### Setup

Clone this repository to your desired folder:

Run this commands:

```sh
  cd my-folder
  git clone https://github.com/iankiema/Bookstore-Capstone.git
```

### Install

Install this project with:

Run this command:

```sh
  cd my-project
  npm install
```

### Usage

To run the project, execute the following command:

- Open index.html using live server extension or;

Run this  command:

```sh
  npm start
```
